ReadMe Book Swap is an app for book lovers and new readers alike! RMBS provides a space for connecting with readers who want to trade books with others. Offer up a book you love and share it with friends! RMBS is the tiny library on the corner in your neighborhood if it was an app! Come search and find who has a book that you would like to borrow, barter, or lend.

Authentication and authorization are done using JSON Web Tokens and Passport and acces to the app can be gained by creating a local account or using your Google account(OAuth 2.0)

The app is build around React.js/Redux on the client side and Node.js on the server side and using other libraries and modules like Bootstrap, React-Bootstrap, Express, Passport.

The database is a SQL database hosted on ElephantSQL and the database logic and data manipulation are being done using Sequelize.
